I’ve added a new FastAPI endpoint at `/greet` that returns “Good morning,” “Good afternoon,” or “Good evening” depending on the server’s current local hour. I also updated the README to document the new route.

---

### 1. New `/greet` endpoint in **app/main.py**

```python
@app.get("/greet")
def greet():
    """
    Returns a greeting message based on the current server time.
    """
    from datetime import datetime

    hour = datetime.now().hour
    if hour < 12:
        message = "Good morning"
    elif hour < 18:
        message = "Good afternoon"
    else:
        message = "Good evening"

    return {"greeting": message}
```
